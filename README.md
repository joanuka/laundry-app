# 🧺 Laundry Management App

A modern and user-friendly mobile application built with **React Native** and **Expo** to simplify laundry service management. The app provides an intuitive interface that allows users to register, log in, navigate through the application, and manage laundry service requests with ease.

---

## ✨ Features

- 🔐 User Authentication (Login & Sign Up)
- 🏠 Home Dashboard
- 🧺 Laundry Service Management
- 🧾 Receipt Generation
- 🚀 Splash Screen
- 📱 Smooth Navigation
- 🎨 Clean and Responsive User Interface

---

## 🛠️ Technologies Used

- React Native
- Expo
- JavaScript (ES6+)
- React Navigation

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/joanuka/laundry-app.git
```

Navigate into the project folder:

```bash
cd laundry-app
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npx expo start
```
